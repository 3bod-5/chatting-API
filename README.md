# basic-Task-manager

##About this project
- As a user you can add,delete a tesk and you can track it by change its status if its done then it marked as Success \ Deadline \ On-progress

## 🚀 Features
- User authentication (JWT)
- sending and reciving messages
- updating User status [online,offline]

## 🛠️ Tech Stack
- Node.js
- Express
- MongoDB
- Socket.io

## 📦 Installation
- npm install
- npm run start
